Password Generator - NET2008 Assignment 4

This Python application generates a secure random password based on user input.
The user chooses the password length, and the program returns a random password
containing letters, digits, and symbols.

How to run locally:
python app.py

How to run using Docker:
docker build -t password-generator-app .
docker run -it password-generator-app
